◆ l_dq

#define l_dq   27

Number of particle velocity vectors in the discretized particle velocity set.

When N_DIM = 2, this is equal to 9 (i.e. D2Q9). In 3D, this value is set to 19 by default (i.e. D3Q19) but can be set to 27 for D3Q27 as well.

◆ l_dq_max

#define l_dq_max   27

Number of particle velocity vectors requried for complete connectivity.

This value is equal to l_dq in 2D but equal to the D3Q27 amount in 3D (whereas l_dq can take on separate values for D3Q19 and D3Q27).
